# 요구사항 (pGO)

## 범위

1. **데이터 업로드 웹** (`dataupload.html`)
   - 지도 배경, 현재 위치 표시.
   - 갤러리에서 이미지 선택 + 문구 입력.
   - 등록 시 현재 좌표와 함께 Firebase에 저장.

2. **플레이 웹** (`getImage.html`)
   - 접속 시 **Go** 버튼 후 지도 표시.
   - 업로드된 스폰 근처 접근 시 후면 카메라 + 이미지·텍스트 오버레이.
   - 터치로 수집; 수집한 스폰은 다시 나타나지 않음.
   - **익명 로그인** 자동 처리.
   - **수집 목록** UI.

3. **플랫폼**
   - 모바일 브라우저(Android / iOS).

4. **배포**
   - Firebase Hosting, 경로: `/pGO/…`.

## 수용 기준 (검증 포인트)

- [ ] `firebase-config.js` 설정 후 익명 로그인 오류 없이 두 페이지가 동작한다.
- [ ] 업로드 후 Firestore `spawns`에 문서가 생기고 Storage `uploads/{uid}/…`에 파일이 있다.
- [ ] 플레이 지도에 미수집 스폰이 마커로 보인다.
- [ ] 스폰 반경(기본 30m) 이내에서 카메라 화면과 오버레이가 뜬다.
- [ ] 탭 후 `users/{uid}/caught/{spawnId}`에 기록되고 해당 스폰 마커가 사라진다.
- [ ] 수집 목록에 캡션·썸네일·시간이 표시된다.

## 비범위 (1차)

- 월드 고정 AR(WebXR 품질 보장), 백그라운드 위치 추적, 스팸/신고 운영 도구.
